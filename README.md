# Relay

Relay is an AI-assisted production workspace for scripted, source-based YouTube channels.

The rebuild is starting with the riskiest product loop: a creator asks a coding agent to build a reusable channel video component, the platform validates it independently, the creator reviews it at arbitrary frames, and the exact approved version renders into video.

## Status

Foundation work is in progress under Linear project [Relay Rebuild — AI-First MVP](https://linear.app/media-pipeline/project/relay-rebuild-ai-first-mvp-5f57317e6c9d).

There is no runnable application yet. Scaffolding is tracked by MED-130. Do not invent temporary setup commands before that issue establishes them.
